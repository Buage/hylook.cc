## Project Description

hytale username lookup. made easy. simple, efficient, beatiful and fast, with an ai generated rarity analysis. free, open source, with an open api.

## Key Components

you can use the hylook api in your projects, without api key. you just have to call it like this

`https://lmao.buage.dev/lookup.php?user={username-here}`

## Key Features

here are some features hylook.cc has

*   efficient username lookup
*   privacy first, no data collection
*   anonymous analytics
*   free to use api
*   ai generated rarity analysis

## Tech Stack

**Frontend**: HTML, CSS, JS **Backend**: PHP
